LLM 與 AI/ML 課程

16 週系統化學習路徑｜從基礎到進階完整教學
打造 AI 時代的核心競爭力

關於本課程

本課程專為想要深入學習大型語言模型（LLM）和人工智慧/機器學習的學習者設計。 從基礎數學和 Python 開始，逐步進階到 LLM 應用開發、微調和部署。 通過系統化的學習路徑，你將掌握從理論到實踐的完整 AI/ML 技能。

課程特色

  • 理論與實踐結合
  • 項目驅動學習
  • 最新技術追蹤
  • 完整技能樹培養

學習時間

  • 建議每週 10-15 小時
  • 理論學習 3-4 小時
  • 程式實作 4-5 小時
  • 專案練習 3-4 小時

適合對象

  • 有 Python 基礎的開發者
  • 想轉型 AI 的工程師
  • 對 LLM 感興趣的學生
  • 需整合 AI 的 PM

學習成果

  • 建立 ML Pipeline
  • 開發 LLM 應用
  • 微調語言模型
  • 部署 AI 服務

技術棧

Python PyTorch TensorFlow HuggingFace LangChain OpenAI API LlamaIndex FastAPI Docker NumPy Pandas Scikit-learn

課程大綱

第一階段：基礎準備 (Week 1-4)

建立 AI/ML 的數學基礎和機器學習核心概念，掌握深度學習基本原理。

Week 1
Python 與數據處理基礎
NumPy、Pandas、Matplotlib、Jupyter Notebook、探索性資料分析
已完成
Week 2
機器學習數學基礎
線性代數、微積分、機率統計、最佳化、梯度下降
已完成
Week 3
機器學習入門
監督式學習、非監督式學習、Scikit-learn、模型評估
已完成
Week 4
深度學習基礎
神經網路、反向傳播、PyTorch 基礎、MNIST 實戰
已完成

第二階段：深度學習進階 (Week 5-8)

深入學習 CNN、RNN、Transformer 等現代深度學習架構，掌握 NLP 基礎技術。

Week 5
卷積神經網路 (CNN)
CNN 架構、ResNet、遷移學習、影像分類實戰
已完成
Week 6
循環神經網路 (RNN)
RNN、LSTM、GRU、Seq2Seq、注意力機制
已完成
Week 7
Transformer 架構詳解
Self-Attention、Multi-Head Attention、從零實現 Transformer
已完成
Week 8
NLP 基礎與詞嵌入
Word2Vec、GloVe、BERT、文本分類、情感分析
已完成

第三階段：大型語言模型 (Week 9-12)

深入理解 LLM 原理，掌握 Prompt Engineering、API 開發和模型微調技術。

Week 9
LLM 原理與架構
GPT 系列、自回歸語言模型、預訓練與微調、Tokenization
已完成
Week 10
Prompt Engineering
Few-shot Learning、Chain-of-Thought、ReAct、提示優化
已完成
Week 11
LLM API 與應用開發
OpenAI API、LangChain、RAG、智能問答系統
已完成
Week 12
LLM 微調技術
LoRA、QLoRA、PEFT、指令微調、領域適配
已完成

第四階段：進階應用與部署 (Week 13-16)

學習 AI Agent 開發、多模態 AI、模型評估和生產環境部署。

Week 13
LLM Agent 開發
Agent 架構、Tool Use、ReAct、AutoGPT、記憶系統
已完成
Week 14
多模態 AI
CLIP、Stable Diffusion、GPT-4V、Whisper、影像描述生成
已完成
Week 15
LLM 評估與安全
評估指標、幻覺問題、模型對齊、RLHF、安全防護
已完成
Week 16
LLM 部署與優化
模型量化、推理優化、API 服務、監控與日誌、成本優化
已完成

學習建議

  • 動手實作：不要只看不做，一定要親自寫代碼
  • 理解原理：不要只會調用 API，要理解背後的原理
  • 持續學習：AI 領域發展快速，保持學習熱情
  • 做好筆記：記錄重要概念和常見問題
  • 參與社群：加入技術社群，與他人交流學習
  • 完成專案：每週都要完成實戰專案

前置知識要求

  • Python 程式設計基礎（變數、函數、類別、檔案處理）
  • 基礎數學（高中程度數學即可，課程會補充）
  • 基本的 Linux 指令操作
  • Git 版本控制基礎
  • 機器學習經驗（加分但非必要）
